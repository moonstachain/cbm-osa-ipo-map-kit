## 〇、核心：一个原子，三级递归

整套模型只有**一个原子引擎：IPO**（把信息炼成一个判断）。往上每一级都是 IPO 的递归组合：

```
IPO   ── 一个判断的信息引擎（采→工→表）           【原子·base case】
 │  ×3（对 O、S、A 各跑一次）
OSA   ── 把一个「点」想透 = O的IPO + S的IPO + A的IPO   【一格 / 一卡】
 │  × 网格（领域 MECE × 三责任层）
CBM   ── 把一个「领域」铺全 = 一格格 OSA 拼成的棋盘     【一张战略地图】
```

**关键**：OSA 不是"另一种东西"，它**就是 3 个 IPO**；CBM 也不是"另一种东西"，
它**就是一网格 OSA**。同一信息引擎，三尺度自相似递归。

## 一、第一级 · IPO（原子引擎·三五三）

- **I 采集·3 标准**：全 / 真·一手 / 细（"能有逐字稿不说大概"；严格形态=正交基张满全空间：全=不漏、正交=不重）。
- **P 处理·5 层**：碎片→特征→观点→洞察→全局最优。合格=金字塔（1 洞察 ← 2-3 观点 ← 全真细信息）。
  跳层红线：跳 L2=偏见 / 跳 L4=空中楼阁 / 直跳=算命。
- **O 输出·3 标准**：闭环（无结果=0）/ 自动化（AI 替 80-90%）/ 智能化（AI 自复盘迭代）。
- **隐藏第四段**：回流（O→执行→现实反馈→成新的 I）。真实形态 I→P→O→[回流]。

## 二、第二级 · OSA = 3 个 IPO（把一个点想透）

| OSA 的一段 | = 谁的 IPO | I 采什么 | P 怎么炼 | O 出什么 |
|---|---|---|---|---|
| **O 目标** | O 的 IPO | 目标空间/候选集全要素 | 信息五层与约束比较到 L5（≥3 候选） | 一个可量化的全局最优目标 |
| **S 策略** | S 的 IPO | 路径/资源/竞争/周期 | 策略组合 + 技术可行/价值/增长实验 | 一套被真实结果验证的策略 |
| **A 行动** | A 的 IPO | 执行条件/约束 | SMART + Human Gate + 自动化阶梯 | 一串可验收、可回流的行动合同 |

三段不跳层：O 没想透定 S = 拿错靶；S 没经过科学实验就放大 = 假增长；A 没有
SMART/Human Gate 就自动化 = 放大风险。隐藏第四段 = 校验与回流（LOOP 非瀑布）。

### SABC 科学实验映射（v2）

| 等级 | 唯一验收口径 |
|---|---|
| C | 策略想法，尚无实验 |
| B | `technical_feasibility` 通过，至少一次可复现端到端闭环 |
| A | `value_validation` 通过，取得真实用户或经营结果；虚荣指标不能单独通过 |
| S | `growth_validation` 通过，至少三个可比较周期，且可复制、可归因 |

“AI 自驱”不是第四实验阶段，而是 A3：AI 自主规划、运行、复盘并留下真实
`changed_rule` 回执。旧创业四周期里的“原型假设”仍只表示“我是谁”；技术实验一律写
“技术可行性假设 / `technical_feasibility_hypothesis`”。

## 三、第三级 · CBM = 网格 × OSA（把一个领域铺全）

**第一刀·横向到边**：切成不重不漏的子领域（究竟完备），空白领域显式立格标 candidates。
**第二刀·纵向分层**：每子领域切三责任层，每层一张 OSA 卡——
战略层（Direct 定方向·靠人脑关键判断）/ 管控层（Control 控系统·确保不翻车）/
执行层（Execute 做动作·可自动化流水）。

每格 = 一张 OSA 卡 = 3 IPO；一张 CBM 图（N域×3层）= 3N 张 OSA = 9N 次三五三。

## 四、评分与验收（`yuanli-osa-card/v2` 引擎强制执行）

自相似不变量每级复验：**究竟-完备-递进-原子四门** + **H/V/A/E 黄金三角**。
- 每叶 1-5 分 + 证据句 + 置信（🟢真据/🟡推断/⚪公式默认）；
- O=L1–L5、S=C–B–A–S、A=A0–A3 三轴独立，`unassessed` 不换算为 0；
- 引擎区分 declared / supported ceiling / effective，不做自动晋级；
- Gold = `O=L5 AND S=S AND A=A3`，并要求证据真实、当前、无冲突且有人批回执；
- 距 100 分的差距逐条列出 = 验收清单，短板层定"下一战"。

## 五、一句话

> IPO 是原子（一次三五三），OSA 是"想透一个点"（3×IPO），CBM 是"铺全一个领域"
> （领域 MECE × 三层，每格一 OSA）。同一信息引擎，递归三层。三层全满 = 独当一面。
